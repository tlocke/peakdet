Peak Detector

Load in a CSV file with a two columns of figures (time (in the format YYYY-MM-DDThh:mm) and data), using the button below. Then a graph of the data and all the peaks (in yellow) and the valleys (in pink) will be displayed. The sensitivity is controlled by the value of the minimum peak height. The lower the minimum height, the higher the number of peaks. There's an example CSV file that you can use to try it out if you haven't got any suitable data to hand.
